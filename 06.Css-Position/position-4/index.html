Does not scroll. This element is set to fixed. Common issues:

  • The element's width collapses - so you have to add 'width: 100%;' to your CSS.
  • The element covers up other content. You'll have to move that other content, with margin rules or something else.
  • You must indicate where the element will be fixed - the most common calue being 'top: 0;'

Allows you to 'scoot' something around a little bit. This element is shifted 80px t the right. Common issues:

  • Cannot be used to fix major layout issues
  • The element may cover up other content.

Allows you force an element out of the DOM and put it exactly where you want it. Common issues:

  • Can totally collapse the layout of a page.
  • The space an element is moved 'from' is not filled in, it remains blank
  • there's a little red circle in the upper-right corner that illustrates absolute positioning.

Allows you force an element to 'stick' ot the top of the page once it's scrolled to (scroll to this one to see!)

  • requires a width value, or it collapses
  • Requires a 'top' value to know where to stop
